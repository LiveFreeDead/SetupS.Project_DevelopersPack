How to make a ppApp from an ssApp

How to make a ppApp from an ssApp        


Warning! This is is just too easy and may cause a false sense of security to develop after prolonged use of this quick and dirty method.

1.        Preparation.

    • Create a temporary "build" folder. Any name will do.
    • Copy the contents of the the ssApp (i.e., "C:\Program Files") to the "build" folder.
    • Right-click the "build" folder and Sendto "SetupS Builder".
    • Important!: Select ppApp "Build Type". This will determine the .app-filename it needs to save the information to.
    • Click Save (Ctrl+S) -- or use the Save button on the Mini-Toolbar. Notice the Install-to Path changes to the %ppApp% variable ;)

2.        "Assembly" tab

    • Remove the installer from the Assembly box. This step is not necessary except perhaps to keep things tidy.

3.        Build!

    • Check your Build options. Click MenuBar "Builds" (between "Options" & "Tools") and select how you want you want your builds done.
    • Finally, when you're ready, click "Tools, Build (F5)" -- or click the "Build" button on the Mini-Toolbar.

4.        Test

    • To test, simply double click on the .app or .apz file created by ssEditor.
