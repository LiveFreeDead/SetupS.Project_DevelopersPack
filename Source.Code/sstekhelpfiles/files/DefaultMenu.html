Default Menu

Default Menu        


This is one of the most important items that needs attention. This is where shortcuts get created if no advanced StartMenu sorting is selected. For ssApps, this is where the installer or setups software will install its shortcuts even if it is subsequently moved by SetupS later. And finally, this item is required for all shortcuts (Desktop, Quicklaunch, Sendto, etc.) no matter if they are created in the Startmenu or not. 

In other words, do NOT skip this one.


Status: Required.
