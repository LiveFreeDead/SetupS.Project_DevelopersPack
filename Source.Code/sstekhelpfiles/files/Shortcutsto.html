Shortcut(s) Location(s)

Shortcut(s) Location(s)        


All Users  - Create shortcut(s) for "AllUsers" instead of "Current User" using the flags below.

Start menu (normal) - Create Shortcuts in the Start Menu. Affected by the Shortcut Options: ''Keep in folder'' and ''Keep all''.

Root - Make a shortcut in the Root of the Start Menu. Only the first shortcut from ''Shortcut Names to Keep'' or from ''Definitions''.

Programs - Make a shortcut in the Programs group of the Start Menu. Only the first shortcut from ''Shortcut Names to Keep'' or from ''Definitions''.

Startup - Make a shortcut in the Startup group of the Start Menu. Only the first shortcut from ''Shortcut Names to Keep'' or from ''Definitions''.

Pin - (Windows 7 only) Pins a shortcut to the Start Menu. Only the first shortcut from ''Shortcut Names to Keep'' or from ''Definitions''.

Desktop - Makes Desktop Icons. Only the first shortcut from ''Shortcut Names to Keep'' or from ''Definitions''.

Quick launch - Makes Quick Launch Icons. Only the first shortcut from ''Shortcut Names to Keep'' or from ''Definitions''.

SendTo - Create a Shortcut in the Send To Menu. Only the first shortcut from ''Shortcut Names to Keep'' or from ''Definitions''.

Taskbar Pin - (Windows 7 only) Pins a shortcut to the Taskbar. Only the first shortcut from ''Shortcut Names to Keep'' or from ''Definitions''.
